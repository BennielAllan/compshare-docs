# 如何发布社区镜像
## 操作说明

本文档旨在指导用户如何在算力共享平台上制作和发布自己的社区镜像。社区镜像是经过自定义配置的操作系统镜像，可以包含预安装的软件、工具或框架，可方便快速部署和使用应用资源。

## 操作须知
1.算力共享平台规定已实名的个人和高校用户可发布定价镜像至社区盈利，企业认证用户可发布镜像至社区但不可参与镜像定价和返利；[去实名去认证](https://console.compshare.cn/uaccount/authentication)   

2.了解Compshare优云智算平台各镜像类型区别：  

基础镜像：底层是容器架构，可选择操作系统框架，**基础镜像可发布至社区，成为社区镜像**；

系统镜像：底层是虚机架构，仅为裸机纯净环境，支持windows和ubuntu，**不可发布至社区**；

社区镜像：已发布至社区的镜像，基于基础镜像修改制作成应用镜像后，可自行发布至社区，在社区镜像模块展示；

私有镜像：可以是基础镜像或系统镜像或社区镜像制作并保存的镜像，创建GPU实例是可从私有镜像中选择；

## 步骤详解
### 第一步：设置平台昵称

点击我的资源→账号管理→账号安全→修改称呼，[点击修改](https://console.compshare.cn/uaccount/userinfo)   

![修改昵称](
https://www-s.ucloud.cn/2024/12/25ab12ef30c95156140e9e51dabc497e_1735642306464.png)

### 第二步：部署GPU实例
1. 登录算力共享平台，选择“部署GPU实例”,[一键部署](https://console.compshare.cn/light-gpu/resources/create)；   

2. 点击“平台镜像”，选择“基础镜像”，*系统镜像暂不支持发布至社区，请一定选择基础镜像* ;

3. 右侧选择GPU卡数量，配置，及磁盘配置；   

4. 选择防火墙（管理服务器端口），如无端口额外需求，用默认的即可；

![部署实例](
https://www-s.ucloud.cn/2025/01/416fdadb228336c1584144ecdf92ece2_1735799981345.png)



### 第三步：进入控制台
#### 启动实例
1. 进入控制台 →实例列表→选择您所创建的实例→登录实例；
2. 登陆实例，安装和配置镜像相关的软件应用；  

![登录实例](
https://www-s.ucloud.cn/2025/08/623977cebd26852052bf92977470fe48_1755018731763.png)

#### 优化和清理
```
1. sudo apt autoremove -y 
2. sudo apt clean
```
### 第四步：保存成私有镜像
1. **请保持实例开机状态下，制作镜像**；

2. 制作镜像 

**入口一：从实例列表中制作镜像**  

![制作镜像](https://www-s.ucloud.cn/2025/08/ce44c4896920bb31a7fc34455cbf9a74_1755050233138.png)

**入口二：从镜像列表中制作镜像**

![制作镜像](https://www-s.ucloud.cn/2025/08/4aae61b75eba76a626f09e7d092c4b48_1755050233148.png)

### 第五步：私有镜像可行性验证
1. 制作出的自制镜像可在 控制台-镜像列表 查看；

2. 可通过自制镜像启动一个新的实例，验证软件安装和配置是否符合预期。

***具体步骤为：部署GPU实例→平台镜像→私有镜像→选择自制镜像→立即部署***；   

![镜像验证](
https://www-s.ucloud.cn/2025/01/50cc5262595210db932d93f19bc9443e_1735801441262.png)

### 第六步：公开镜像发布至社区
#### 在页面顶部找到“发布社区镜像”按钮，点击进入
1. 设置按照步骤分别填写镜像名称，填写镜像的描述信息，包括用途、预装软件等信息。（镜像定价的最小加价单位是0.01元）    
 
2. 填写完成后选择确认，镜像审核预计在3个工作日完成。

![镜像发布1](https://www-s.ucloud.cn/2025/06/1f742baf18dc6d3acf6ad686f0b5df8d_1749612661290.png)

### *注意事项*
**1. 确保镜像中不包含敏感信息，如密码或密钥；**   

**2. 遵守算力共享平台的使用条款，不上传违反版权或法律法规的内容；**   

**3. 镜像制作可能需要一些时间，请耐心等待；**
