# 创建GPU实例     
您可以通过平台镜像和社区镜像可以快速创建实例，不用自行从零开始搭建运行环境，极大程度地提高了执行效率，减少部署环境所需的时间。    

## 通过平台镜像创建实例   

1. 平台内置了许多不同的基础镜像和系统镜像，您可以按照运行所需的环境选择相应的镜像快速创建实例；     

2. 这里以创建CUDA基础镜像为示例，您可以通过点击镜像的下拉框，选择相应的CUDA版本：   

![创建](https://www-s.ucloud.cn/2025/08/20feeb4df7890250afb73e06d94644a8_1755018731757.png)    

3. 同时，在页面右侧按需选择GPU型号、数量等配置，选择付费模式，点击立即部署创建实例：    

> 这里推荐使用RTX40系GPU部署实例以获得更好的效果。   

![创建](https://www-s.ucloud.cn/2025/08/3e48cf3c8d12e2ecb50e42fb9c52dfca_1755018731760.png)      
    

4. 稍等片刻后会自动创建并启动实例，通过实例列表可以查看实例的运行状态，支持随时关闭或启用实例：

![创建](https://www-s.ucloud.cn/2025/08/623977cebd26852052bf92977470fe48_1755018731763.png)      
 

##  通过社区镜像创建实例   

1. 首先，您可以在镜像社区查看到专业镜像作者或用户分享到社区的镜像：   

![镜像社区](https://www-s.ucloud.cn/2025/04/dfab6926425be199b0fd821949c6709b_1745734705581.png)     

2.  点击您需要的镜像进入镜像详情页，通过页面右侧的使用该镜像创建实例可以快速创建一个实例：  

![创建](https://www-s.ucloud.cn/2025/04/c09ee583c50550116449f22b75648980_1745734808087.png)    

3.  接着将跳转到配置页面，按需选择GPU型号、数量等配置后点击立即部署： 

> 这里推荐使用RTX40系GPU部署实例以获得更好的效果。   

![创建](https://www-s.ucloud.cn/2025/08/15745464f1c769728eb3094c05cd141b_1755018731768.png)     

4. 稍等片刻后会自动创建并启动实例，通过实例列表可以查看实例的运行状态，支持随时关闭或启用实例： 

![创建](https://www-s.ucloud.cn/2025/08/623977cebd26852052bf92977470fe48_1755018731763.png)
