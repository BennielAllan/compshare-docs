# Windows服务器远程登录操作手册   
适用于系统镜像（虚机）

## 远程桌面登录   
1. 打开本地电脑中的【远程桌面连接】应用，红框处输入创建主机后分配的外网IP 地址，点击【连接】；   

![图片](https://www-s.ucloud.cn/2025/07/0a6c313d6b4c399e0191582838511175_1753251106986.jpg)   

2. 【输入你的凭据】输入主机的登录名和密码，登录名为administrator, 密码可从控制台复制，即可远程登录到创建的Windows云主机中；   

![图片](https://www-s.ucloud.cn/2025/07/8b5cf91f2ec786546221d6b96d34b95a_1753251256630.jpg)   

## 关闭IE增强设置   
应用场景：首次登录Windows云主机，打开IE 浏览器提示网站内容被阻止，需要关闭IE增强的安全配置。   

![图片](https://www-s.ucloud.cn/2025/07/4ba057d9d3777e00cc9b48291fd246b7_1753251425786.jpg)    

1. 单击云主机中的Windows菜单键，选择【服务器管理器】   

![图片](https://www-s.ucloud.cn/2025/07/efbfe5367dccae3dbab85060d2d6fe5c_1753251477328.jpg)  

2. 单击左侧【本地服务器】,选择IE增强的安全配置的【启用】按钮   

![图片](https://www-s.ucloud.cn/2025/07/52f144d385a3c2b57083591e1f4a1d78_1753251541064.jpg)    

3. 【IE安全性等级设置】}打开IE浏览器，需要设置【安全】选项下，安全性等级降至【中高】   

![图片](https://www-s.ucloud.cn/2025/07/6db68cadecb8618bbfac27bee659f20a_1753251603793.jpg)    

**修改完成后，可使用IE前往下载并安装Chrome浏览器**    

## 其他服务器设置   
### 取消远程登录时间限制   
应用场景：远程登录一段时间后没使用，超过远程登录会话时间，登录会卡死，导致需要重新断开再登入    

1. 键盘点击【win+R】弹出运行，输入【gpedit.msc】   

![图片](https://www-s.ucloud.cn/2025/07/6c5d833b184463cd119954090455a842_1753251772190.jpg)    

2. 找到【计算机配置】 →【管理模板】 →【Windows组件】 →【远程桌面服务】 →【远程桌面会话主机】 → 【会话时间限制】   

![图片](https://www-s.ucloud.cn/2025/07/c11e1a4cf1176924aaf7fbab495d7917_1753251821145.jpg)    

3. 单击【设置已中断会话的时间限制】,再点击【策略设置】,选择【已启用】,选项选择【从不】,点击【确定】确认   

![图片](https://www-s.ucloud.cn/2025/07/7ff4796c2b63bb7afd6139f5d3f9b9ab_1753251870707.jpg)    

4. 单击【设置活动但空闲的远程桌面服务会话的时间限制】,再点击【策略设置】,选择【已启用】,选项选择【从不】,点击【确定】确认   

![图片](https://www-s.ucloud.cn/2025/07/3594c017c7b5dd6a55a57e6ce919b294_1753251917152.jpg)   

### 本地文件上传设置   
应用场景：通过云主机访问本地电脑文件，方便资料上传和管理。   

1. 打开【远程桌面连接】,选择左下方倒三角【显示选项】   

![图片](https://www-s.ucloud.cn/2025/07/eedd0d0eda50fcea233b3a0dcb6848c6_1753252043978.jpg)     

2. 选择【本地资源】 → 【本地设备和资源】 → 【详细信息】   

![图片](https://www-s.ucloud.cn/2025/07/2e8dd4f8e5bca5b7265866b25a3131f5_1753252098782.jpg)    

3. 勾选】需要云主机中需要访问的本地磁盘，点击【确定】   

![图片](https://www-s.ucloud.cn/2025/07/e070218dca77339a3d2cac08ce9779a0_1753252145510.jpg)    

4. 重新远程连接云主机，就可以看到共享的本机磁盘了，轻松实现文件拷贝等操作   

![图片](https://www-s.ucloud.cn/2025/07/0f24a53fcb91caae4b707d48e67ed3c4_1753252196103.jpg)
