# 套餐用量统计

本页说明「当前套餐」页面各区域的含义，以及如何排查单次调用情况。

## 1. 入口

控制台左侧导航：「Agent Plan」→「套餐管理」
> 如您购买过已下架的旧版套餐，页面将默认显示旧版用量看板，顶部提示区域提供新版 Coding Plan 入口。
![入口](https://www-s.ucloud.cn/2026/04/23e697ca955908ef422b91ba175b428d_1777014442867.png)
---

## 2. 套餐卡片

页面顶部显示当前生效的套餐：

| 字段 | 说明 |
|------|------|
| 套餐 ID | 唯一标识，排查时最重要 |
| 有效期至 | 套餐到期时间 |
| 操作按钮 | 「续费」或「升级」 |

### 调用额度进度条（3 个 Tab：5小时 / 本周 / 本月）

进度条分 3 个 Tab 展示，各自独立计量

![进度条](https://www-s.ucloud.cn/2026/04/7fda5142ee38a94bec60ec50cbe56ec6_1777014442839.png)

---

## 3. 调用明细 Tab

查看每次单条请求的消耗情况。

**筛选维度：** 最近 1 小时 / 1 天 / 14 天 / 30 天 / 自定义

| 列名 | 说明 |
|------|------|
| 模型 | 本次请求使用的模型 |
| Key ID | 使用的 API Key |
| 套餐 ID | 实际扣除的套餐 |
| 请求消耗次数 | 一般为1，部分模型倍率为2或3 |

---

## 4. 常用排查方法

| 排查需求 | 操作方法 |
|---------|---------|
| 确认某次调用扣了哪个套餐 | 调用明细 Tab → 查套餐 ID 列 |
| 查套餐到期时间 | 套餐状态卡片 |
| 查月度剩余额度 | 套餐状态卡片 |
