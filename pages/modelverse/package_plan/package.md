# 套餐包快速上手

本页按界面路径说明：如何找到套餐入口、查看规则、完成购买。

## 1. 进入「套餐订阅」页面
![套餐入口](https://www-s.ucloud.cn/2026/05/1b36bbf629a36e18067f0c84757a5450_1778477448933.png)

操作路径：

1. 顶部导航点击「模型 API」
2. 左侧导航点击「Agent Plan」→「套餐订阅」
3. 进入购买页，选择不同的套餐包创建

---

## 2. 先看「适用模型说明」
![适用模型](https://www-s.ucloud.cn/2026/05/a7c903082f4f0586bbf071e69ecf3670_1778477448944.png)

每个套餐卡片右下角有「查看适用模型」弹窗，内含：
- **限流说明**：最大并发数
- **支持模型列表**：套餐内可用的全部模型及调用规则

---

## 3. 套餐类型说明

### 套餐优势

- 一个套餐涵盖 DeepSeek、Kimi、GLM、MiniMax等主流编程模型，完整列表见「规则说明」
- 使用专属 API Key，独立于普通按量 Key
- 支持 Claude Code、Trae、Openclaw、Hermes 等主流 AI 工具接入
- 支持 OpenAI 兼容协议与 Anthropic 兼容协议

## 4. 购买与查看
1. 选择套餐，点击「立即购买」完成支付
2. 支付完成后套餐自动激活，生成专属 API Key
![专属KEY](https://www-s.ucloud.cn/2026/05/42de887fd86f77ca326d26c579b4b59b_1778477448952.png)
3. 在「Coding Plan → 当前套餐」复制专属 Key 及 Base URL，配置到编程工具
4. 在「当前套餐」查看调用额度与明细

> **注意**：订阅服务一经购买即视为确认，不支持退款。建议根据实际使用频率选择合适档位。
