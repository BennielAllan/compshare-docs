# OpenClaw 云端服务

优云智算提供一键云端部署的 OpenClaw 服务，含专属计算沙箱资源，7×24 在线，支持零门槛使用及无缝调用 Coding Plan 套餐内的模型。

---

## 开通服务

订阅 **¥199/月 Pro** Coding Plan 套餐后，即可开通 OpenClaw 云端服务（每个 Pro 套餐支持创建 1 个 OpenClaw）。

> **重要提示：** 套餐到期后，您创建的 OpenClaw 将被删除，其中的数据将为您保留 **24 小时**，请及时备份。

---

## 创建并使用 OpenClaw

1. 在左侧导航选择「Coding Plan」→「OpenClaw」
2. 选择要绑定的 Key，点击「创建服务」
3. 等待约 3 分钟完成初始化
4. 创建完成后即可通过浏览器直接访问，无需本地安装任何环境

### Key 选择说明

创建或使用 OpenClaw 时，右上角支持选择两种 Key：

| Key 类型 | 效果 |
|---------|------|
| **套餐专属 Key** | 模型列表过滤为套餐内支持的模型，调用走套餐额度 |
| **模型广场 Key** | 老套餐用户可选择模型广场key，模型列表为老套餐支持模型 |

注意：如需按量使用原价模型和Key，也可在对话框内用自然语言告诉openclaw，进行模型替换
---

## 功能支持

| 功能 | 说明 |
|------|------|
| 查看并配置技能 | 支持查看、安装技能（通过对话安装） |
| 查看并配置定时任务 | 支持设置定时任务，主动发送提醒 |
| 配置消息渠道 | 支持集成企业微信、飞书、钉钉 |
| 模型切换 | 一键切换套餐内所有支持模型 |

---

## 注意事项

- OpenClaw 仅随 Pro 套餐提供，Lite 套餐不包含此权益
- 使用套餐专属 Key 时，每次调用消耗套餐额度 1 次（部分套餐是多次）
- 套餐到期 → OpenClaw 删除 → 数据保留 24 小时，逾期数据不可恢复
- 续费或升级套餐后，OpenClaw 服务可继续使用，无需重新创建
