# 在ChatHub中使用优云智算LLM API
## 关于ChatHub   
ChatHub是一款聊天机器人聚合客户端，让你在一个应用里轻松切换 ChatGPT、Google Gemini、Claude 3、Bing Chat、Perplexity 等主流模型，还能接入更多开源机器人。无论是快速对比回答，还是搞定复杂任务，ChatHub 都能轻松满足。    

## 在ChatHub中使用LLM API   
### 第一步：[获取 API Key](https://console.compshare.cn/light-gpu/api-keys)       

### 第二步：在ChatHub进行配置  
1. 在 Chrome 应用商店下载 ChatHub插件   

![](https://www-s.ucloud.cn/2025/08/23cf52b10fe2302d92f6639a703643d0_1754278974769.png)  

2. Chathub的配置

1) 进入Chathub。点击设置，下拉到自定义聊天机器人，点击 ” Add ”，添加自定义聊天机器人。   

![](https://www-s.ucloud.cn/2025/08/b8bddbc6e552f8635ac5afdd3b971909_1754279088321.png)   

2) 自定义聊天机器人的设置   
- Chatbot Name：自定义名称，如：优云智算
- API Host：`https://api.modelverse.cn/v1/chat/completions`
- API Key：粘贴在优云智算创建的Key
- Model：设置为创建的API模型名，如：deepseek-ai/DeepSeek-R1-0528
- 点击 ”Confirm“ 保存设置

![](https://www-s.ucloud.cn/2025/08/e48f03b5c50d28f0c172c88bde9da924_1754279156919.png)    

3. 开始对话，在侧边栏的自定义模型列表中，选择 ”优云智算“ 启动对话 
  
![](https://www-s.ucloud.cn/2025/08/a39f7b55a9330c4689ca910184a45b2a_1754279211087.png)
