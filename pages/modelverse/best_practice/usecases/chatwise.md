# 在ChatWise中使用优云智算LLM API 
## 关于ChatWise   
ChatWise 是一款简洁且功能强大的本地 AI 聊天工具，支持多种大型语言模型，如 OpenAI、Claude、Gemini、DeepSeek 和 Ollama 等。其支持本地数据存储、多模态交互以及强大的性能优化等，ChatWise 为用户提供了安全、便捷且功能丰富的智能聊天体验。   

## 在ChatWise中使用LLM API   
### 第一步：[获取 API Key](https://console.compshare.cn/light-gpu/api-keys)    

### 第二步：在ChatWise进行配置   
1. 配置 ChatWise，前往ChatWise官网下载应用：https://chatwise.app/  

![](https://www-s.ucloud.cn/2025/08/f847ba99e4ef35a41ff2fce4df3505fd_1754286702707.png)  

2. 打开应用，左下角打开“设置”   

![](https://www-s.ucloud.cn/2025/08/6e238b9133777d4b65a4d6c7fe471bd5_1754286738099.png)  

3. 进入设置后，选择“供应商”，点击左下角“+”符号，选择“OpenAI Compatible”   

![](https://www-s.ucloud.cn/2025/08/25d853794144474855c46589810e74b4_1754286771748.png)    

4. API Key的配置，输入模型供应商名字、Base URL、Key后点击“获取模型列表”   

![](https://www-s.ucloud.cn/2025/08/bf79d9d0f3c385514687135634aef14c_1754286806919.png)    

5. 能够看到所有支持的优云智算模型，点击“全选”，再点击“确认”，即可添加所有支持的模型   

![](https://www-s.ucloud.cn/2025/08/ae1b61cb9709f8495da857dde06c30e3_1754286839343.png) 

6. 回到主页，即可查看已添加的模型并选择开始与模型对话   

![](https://www-s.ucloud.cn/2025/08/878bf22d06ed493a540e47aaf61ba017_1754286880791.png)
