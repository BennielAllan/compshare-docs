# 在AnythingLLM中使用优云智算LLM API  
## 关于AnythingLLM    
AnythingLLM是一款专为本地化部署设计的工具，支持多种大模型 API 接入。它的核心优势在于：
- 免费开源：无需额外付费，直接使用。
- 简单易用：无需翻墙，国内用户也能轻松上手。
- 功能强大：无缝集成本地文件，支持多任务处理，让您的效率直接起飞。

## 在AnythingLLM中使用LLM API   
### 第一步：[获取 API Key](https://console.compshare.cn/light-gpu/api-keys)    

### 第二步：进入AnythingLLM进行配置   
1. 安装 AnythingLLM 软件：
首先，您需要在AnythingLLM官网：https://anythingllm.com/desktop 下载并安装软件。安装完成后，打开软件，准备进行后续配置。

![](https://www-s.ucloud.cn/2025/08/8b7059383ec9bee77e4d4d88b781de9d_1754278536295.png)   

2. 配置 AnythingLLM 设置   

1） 打开 AnythingLLM 软件，点击左下方设置图标进入设置界面  

![](https://www-s.ucloud.cn/2025/08/715e0107f0119fa4b066c80bfe93a8f8_1754278620145.png)   

2） LLM首选项的设置   

- 在LLM提供商选项中，选择Generic OpenAI；
- 在API Key栏中，粘贴从优云智算生成的API Key；
- 在Base URL栏中，填写：`https://api.modelverse.cn/v1` ；
- 在Model ID栏中，粘贴你从优云智算选择的模型ID，如：deepseek-ai/DeepSeek-R1-0528  

![](https://www-s.ucloud.cn/2025/08/24a58356f64048a83af05e47f73c077c_1754278671097.png)  

3. 尝试提问，你可以尝试在 AnythingLLM 中提问，默认会使用在设置里面调用的API模型   

![](https://www-s.ucloud.cn/2025/08/be8509f01a12d48c371dd0f8521e5b38_1754278739291.png)
