# 在纳米AI客户端中使用优云智算LLM API     
## 关于纳米AI   
纳米AI是一个开源免费的自动化工作流平台，它提供了 200 多个不同的节点来自动化工作流程，通过编排，可以实现跨不同服务的自动化流程。   

## 在纳米AI客户端中使用LLM API   
### 第一步：[获取 API Key](https://console.compshare.cn/light-gpu/api-keys)     

### 第二步：在纳米客户端中进行配置   

1. 下载安装纳米AI客户端：https://bot.n.cn/download?src=ec_ucloud

2. 选择云厂商 - 添加模型

![nami](https://www-s.ucloud.cn/2025/08/f9bba5bf5f36335eade2c990d3c200a7_1755152612266.png)

3. 选择优云智算，添加模型API Key，检测可联通

![nami](https://www-s.ucloud.cn/2025/08/41713b037c4bb23a54b1acb50a17cf87_1755152612285.png)


3. 在对话界面中选择模型对话

![nami](https://www-s.ucloud.cn/2025/08/4ad5693a581f7c176f0057e17684f90d_1755152612298.png)
