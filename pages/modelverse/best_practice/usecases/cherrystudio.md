# 在Cherry Studio中使用优云智算LLM API  
## 关于Cherry Studio   
Cherry Studio 是一款多模型桌面客户端，目前支持：Windows 、Linux 、MacOS 系电脑安装包。它聚合主流 LLM 模型，提供多场景辅助。用户可通过智能会话管理、开源定制、多主题界面来提升工作效率。   

## 在Cherry Studio中使用LLM API   
### 第一步：[获取 API Key](https://console.compshare.cn/light-gpu/api-keys)   

### 第二步：进入Cherry Studio进行配置    
1. 进入Cherry Studio后，先点击左下角设置，自定义提供商名称如：优云智算，点击“确认”     

![](https://www-s.ucloud.cn/2025/08/252e8f3f6dc2d38926f05bc938088034_1754278012584.png)   

2. 复制在优云智算中创建的API Key；输入API地址：`https://api.modelverse.cn` ，再点击 “添加”  

![](https://www-s.ucloud.cn/2025/08/9f96d2252ff835f8388e4bd3bf331d65_1754278103545.png)  

3. 填写在优云智算中建立的API的模型名，如：deepseek-ai/DeepSeek-R1-0528   

![](https://www-s.ucloud.cn/2025/08/e2da89fc8e2127b4fb399f848e87fd96_1754278144777.png)  

4. 点击”检测“，检测模型是否连接成功   

![](https://www-s.ucloud.cn/2025/08/e2b95ed5b5c989faf82288a8c6a4d26b_1754278180586.png)  

5. 点击对话框中的 “@选取刚刚设置好的模型”     

![](https://www-s.ucloud.cn/2025/08/f5c7fc812e1aa28841fb5eb47f3dbacd_1754278232323.png)
