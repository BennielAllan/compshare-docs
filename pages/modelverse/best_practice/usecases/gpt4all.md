# 在GPT4All中使用优云智算LLM API  
## 关于GPT4All   
GPT4All 是由 Nomic AI 开发的一个开源项目，旨在让用户能够在个人设备上本地运行大型语言模型（LLM），无需依赖云服务。   

## 在GPT4All中使用LLM API   
### 第一步：[获取 API Key](https://console.compshare.cn/light-gpu/api-keys)   

### 第二步：在GPT4All进行配置    
1. 下载GPT4All，前往GPT4All官网进行下载并安装：https://www.nomic.ai/gpt4all   

![](https://www-s.ucloud.cn/2025/08/c12ec179f38188debc5b78749ddbb151_1754287023428.png)  

2. GPT4All的配置   

1）进入GPT4All，点击“查找模型”   

![](https://www-s.ucloud.cn/2025/08/a9e381df1b71b11452f2d3e09eb6ac78_1754287087308.png)  

2）选择“自定义”，输入API密钥，基础URL：`https://api.modelverse.cn/v1`，模型名称如：deepseek-ai/DeepSeek-R1-0528。再点击安装 

![](https://www-s.ucloud.cn/2025/08/7765b1716906a66d076c2142c8e114a8_1754287122112.png)    

3）API设置完成后。进入对话，选择添加的API模型即可实现对话   

![](https://www-s.ucloud.cn/2025/08/dfad207e4e4bd9c53ed70d218b5a6c0e_1754287158418.png)
