# 会员规则

平台实行标准定价，不同会员等级用户可享受不同的权益折扣，最低支持9折，GPU与模型API产品均适用。

## 会员等级与权益

| 会员等级 | 达成条件 | 折扣优惠 | 一次性赠金 | GPU实例配额 | 技术支持 |
| --- | --- | --- | --- | --- | --- |
| LV0普通会员 | 完成注册 | 无 | 无 | 0 | 无 |
| LV1青铜会员 | 完成个人实名认证，历史总充值金额=0元 | 无 | 5元 | 1 | 公共社群支持 |
|  | 完成个人实名认证，历史总充值金额＞0元 |  |  | 5 |  |
| LV2白银会员 | 完成高校/企业认证 或 累计消费≥500元 | 9.5折 | 5元 | 20 | 公共社群支持 |
| LV3黄金会员 | 累计消费≥5000元 | 9.3折 | 50元 | 50 | 1v1专属支持 |
| LV4钻石会员 | 累计消费≥10000元 | 9折 | 100元 | 100 | 1v1专属支持 |   

## 等级与GPU配额说明

| 等级 | 描述 | 最大GPU配额 |
|:---|:---|:---|
| LV0（未充值） | 历史总充值金额 = 0 | 1 |
| LV1（青铜/首充） | 历史总充值金额 > 0 且 未达到 LV2 门槛 | 5 |
| LV2（白银） | 累计消费 ≥ 500元 或 完成企业/高校认证 | 20 |
| LV3（黄金） | 累计消费 ≥ 5000元 | 50 |
| LV4（钻石） | 累计消费 ≥ 10000元 | 100 |

**说明：**
- **GPU配额**指当前等级下，可同时创建的机器数量上限。
- 如需提升配额，请完成对应等级的消费或认证要求。

## 会员升级规则
当您的累计消费金额达到下一个会员等级的要求，系统将自动为您升级会员等级。升级后，您将即刻享受新等级对应的会员折扣进行消费。
