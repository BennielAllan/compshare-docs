# CompShare 优云算力文档

本仓库包含 [CompShare 优云算力平台](https://www.compshare.cn) 的全部帮助文档，涵盖：

- **GPU 操作指南** — 创建实例、登录、磁盘、镜像、计费等
- **GPU 服务 API** — 完整的 OpenAPI 参考文档
- **模型 API 服务（Modelverse）** — 文本、图像、音频、视频模型接入
- **账号与账单** — 注册、实名、充值、发票
- **平台概述** — 产品介绍、会员权益、活动公告
- **服务协议** — 用户协议、隐私政策、SLA

## 文档结构

```
pages/
├── gpus/          # GPU 服务 API 文档
├── operation/     # GPU 操作指南
├── modelverse/    # 模型 API 服务文档
├── uaccount/      # 账号与账单
├── overview/      # 平台概述
└── serviceagreement/  # 服务协议
public/            # 文档图片资源
```
